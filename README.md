# Leap Year Checker

A C program created while learning with Exercism.

## What it does
Determines if a given year is a leap year based on these rules:
- Divisible by 400 → Leap year
- Divisible by 100 → NOT a leap year  
- Divisible by 4 → Leap year
- Otherwise → NOT a leap year

## Files
- `leap.c` - Function implementation
- `leap.h` - Function declaration
- `test_leap.c` - Test cases

## Learning Goals
- Understanding boolean algebra
- Using if/else statements in C
- Working with the modulo operator (%)
